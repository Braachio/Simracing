MoTeC CSV 변환 가이드

MoTeC i2에서 주행 데이터를 CSV로 내보내려면 아래 절차를 따르세요:

  1. MoTeC i2를 실행하고 .ld, .ldx 등 로그 파일을 엽니다.
  2. 상단 메뉴에서 File → Export... 을 클릭합니다.
  3. 필요한 설정을 선택합니다.
  4. CSV로 저장한 후, 이 파일을 웹사이트에 업로드하면 분석 리포트를 받을 수 있습니다.

권장 설정

  • Output File Format: CSV File
  • Output Sample Rate: 200 Hz
  • Include Time Stamp 체크
파일 이름은 자유롭게 설정해도 됩니다. .csv 확장자를 유지하세요.
